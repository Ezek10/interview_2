## Descripción

Este repositorio esta preparado para la resolucion de la entrevista tecnica propuesta por ___ para un puesto como Python Developer.
Se puede encontrar las condiciones de la entrevista en los archivos PDF

## Instalación

Clone el repositorio: 

    git clone https://github.com/Ezek10/interview_2.git

Cree una unidad virtual: 

    python -m venv .venv

Activar ambiente virtual:

    .\.venv\Scripts\activate

Instale las dependencias:

    pip install -r requirements.txt
